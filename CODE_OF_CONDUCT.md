# Code of Conduct

This project adopts the [Contributor Covenant][covenant], version 2.1, as its
Code of Conduct. A full copy is available at the link below.

- Covenant text: https://www.contributor-covenant.org/version/2/1/code_of_conduct/
- Translations: https://www.contributor-covenant.org/translations

## Scope

The Code of Conduct applies to all project spaces — this repository, its
issues and pull requests, its discussions, and any related communication
channels operated by the project.

## Enforcement

Reports of unacceptable behavior can be sent to the project maintainers at
`{{SECURITY_CONTACT}}`. All reports are reviewed and investigated promptly
and fairly. Maintainers are obligated to respect the privacy and security of
the reporter of any incident.

Enforcement follows the Community Impact Guidelines published with the
Contributor Covenant v2.1.
